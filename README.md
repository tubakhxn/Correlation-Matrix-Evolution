# Correlation Matrix Evolution

A fully self-contained Streamlit app to visualize the evolution of rolling correlation matrices for synthetic financial assets.

## What is this project?
This project generates synthetic correlated returns for 5 assets, computes rolling correlations, and visualizes the animated correlation matrix using Plotly in a dark-themed Streamlit dashboard.

## How to Fork
1. Click the "Fork" button at the top right of this repository on GitHub.
2. Clone your forked repo:
   ```
   git clone https://github.com/YOUR_USERNAME/Correlation-Matrix-Evolution.git
   ```
3. Install Python 3.11+ and required packages (see below).
4. Run the app:
   ```
   streamlit run correlation_matrix_evolution.py
   ```

## OS Compatibility
- Developed and tested on **Windows**.
- Should also work on macOS and Linux with Python 3.11+ and required packages.

## Requirements
- Python 3.11+
- streamlit
- plotly
